# Security Policy

## Reporting a Vulnerability

Please do not open public issues for suspected security vulnerabilities.

Report privately by email to: `jason@missionmindedmakers.com`

Include:
- a clear description of the issue,
- impact assessment,
- reproduction steps or proof of concept,
- affected files/sections and version context.

## Response Process

- Initial acknowledgement target: within 5 business days.
- We will triage, validate, and coordinate remediation.
- If confirmed, we will publish a fix note once mitigation is available.

## Scope Notes

This repository is currently draft/spec focused; still report issues related to:
- protocol security assumptions,
- signature or verification guidance,
- privacy or data leakage concerns,
- unsafe implementation guidance.
